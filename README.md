功能：
- 对IMGV2格式的信息读取。  
- 指向帧的信息会显示为指向目标的信息。  

---

相关信息：
- 开发平台：Visual Studio 2019
- 项目类型：控制台程序
- 目标框架：.NET Framework 4.5
- 插件依赖：无
